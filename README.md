## What is this repo?
Essentially a python library for bioinformatics stuff. 
The idea is to represent commonly encountered bioinformatics objects (files, databases, sequences, structures) as classes and write functions within these classes to work with the objects.
For example, the class FastaFile represents, obviously, a FASTA file. It contains methods to read the FASTA file and convert it to a python dictionary with the header as key, 
and the value as the sequence. The dictionary can then be used for whatever downstream processing.
